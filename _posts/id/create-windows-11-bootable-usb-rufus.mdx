Melakukan instalasi bersih Windows 11 merupakan langkah tepat untuk mendapatkan sistem yang stabil, ringan, dan bebas dari aplikasi bawaan yang tidak diperlukan. Metode paling efektif untuk melakukannya adalah menggunakan media instalasi berbasis USB.

Salah satu utilitas yang paling andal untuk tugas ini adalah **Rufus** — aplikasi ringan, cepat, dan fleksibel, termasuk kemampuan untuk menyesuaikan persyaratan instalasi seperti TPM 2.0 dan Secure Boot.

Panduan ini akan menjelaskan seluruh proses secara sistematis, mulai dari tahap persiapan hingga USB bootable siap digunakan.

---

## 1. Tahap Persiapan (Wajib Disiapkan)

Sebelum memulai proses pembuatan media instalasi, pastikan beberapa komponen berikut telah tersedia:

### 1.1. USB Flash Drive

Gunakan flash drive dengan kapasitas minimal **8 GB**. Pastikan perangkat dalam kondisi baik.

### 1.2. File ISO Windows 11

Siapkan file installer Windows 11 dalam format `.iso`.

<DownloadButton id="windows-11-iso-24h2" />

### 1.3. Aplikasi Rufus

Gunakan versi terbaru Rufus.

<DownloadButton id="rufus-installer" />

### 1.4. Cadangkan Data Penting

> ⚠️ **Peringatan**
> Proses ini akan memformat flash drive sepenuhnya. Semua data di dalamnya akan terhapus permanen.

---

## 2. Proses Pembuatan Bootable Windows 11

![USB Flash Drive dicolokkan ke laptop](/images/_posts/rufus/rufus-banner.webp)

Setelah semua persiapan selesai, lanjutkan dengan langkah berikut.

---

### Langkah 1 — Jalankan Rufus dan Pilih Perangkat

Buka aplikasi Rufus dan hubungkan flash drive ke komputer. Perangkat akan otomatis terdeteksi pada bagian **Device**.

![Tampilan awal Rufus dengan flash drive terdeteksi](/images/_posts/rufus/rufus-4.9_1.png)

Pastikan perangkat yang dipilih sudah benar sebelum melanjutkan.

---

### Langkah 2 — Pilih File ISO Windows 11

Pada bagian **Boot selection**, klik **SELECT**, lalu pilih file ISO Windows 11 yang telah diunduh.

![Memilih file ISO Windows 11 di Rufus](/images/_posts/rufus/rufus-4.9_2.png)

Setelah file dipilih, Rufus akan mengatur beberapa parameter secara otomatis.

---

### Langkah 3 — Verifikasi Skema Partisi

Pastikan konfigurasi berikut digunakan:

-   **Partition scheme:** GPT
-   **Target system:** UEFI (non-CSM)

Konfigurasi ini sesuai dengan standar instalasi Windows 11 pada sistem modern.

---

### Langkah 4 — Mulai Proses

Klik tombol **START** untuk memulai pembuatan media bootable.

---

### Langkah 5 — Konfigurasi Windows User Experience

Setelah menekan START, jendela konfigurasi tambahan akan muncul.

Centang opsi berikut jika diperlukan:

-   **Remove requirement for 4GB+ RAM, Secure Boot and TPM 2.0**

![Jendela kustomisasi Windows di Rufus](/images/_posts/rufus/rufus-4.9_3.png)

Klik **OK** untuk melanjutkan proses.

---

### Langkah 6 — Tunggu Hingga Selesai

Rufus akan mulai menyalin file instalasi ke flash drive. Proses ini membutuhkan beberapa menit.

![Proses selesai dengan status READY](/images/_posts/rufus/rufus-4.9_4.png)

Ketika status berubah menjadi **READY**, flash drive Anda telah siap digunakan.

---

## Penutup

Media instalasi Windows 11 kini siap digunakan untuk instalasi bersih pada perangkat berbasis UEFI. Dengan konfigurasi yang tepat, proses instalasi akan berjalan lebih efisien dan minim kendala.
