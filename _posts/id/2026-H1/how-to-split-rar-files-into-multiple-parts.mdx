Sebagai blogger yang kadang membagikan file game atau aplikasi berukuran besar, saya sering dapat permintaan yang sama: "Bisa dibagi jadi beberapa part biar download-nya lebih ringan?"

Dan jujur, itu permintaan yang sangat masuk akal.

Dengan file yang dipecah menjadi beberapa bagian, pengunjung bisa unduh satu per satu. Risiko gagal di tengah jalan juga lebih kecil dibanding unduh satu file besar sekaligus.

Selain itu, metode ini juga berguna untuk kebutuhan lain, misalnya:
- Mengirim file lewat email yang punya batas ukuran lampiran.
- Menyimpan file ke beberapa flash drive atau media dengan kapasitas terbatas.

Kabar baiknya, software legendaris **WinRAR** sudah punya fitur ini dan langkahnya sangat mudah.

## Panduan Cepat Menentukan Ukuran Part

Pemilihan ukuran part itu penting. Salah ukuran biasanya bikin proses upload/download jadi lebih ribet.

| Skenario | Ukuran Part yang Disarankan | Alasan |
|---|---:|---|
| Kirim lewat email | `20M` sampai `24M` | Aman untuk batas umum 25 MB setelah overhead metadata. |
| Kirim via aplikasi chat | `50M` sampai `100M` | Lebih gampang retry kalau ada satu upload gagal. |
| Upload cloud dengan internet kurang stabil | `100M` sampai `250M` | Seimbang antara kecepatan dan biaya retry. |
| Pindah file ke flash drive lama | `500M` | Lebih aman untuk media penyimpanan yang kualitasnya beragam. |
| Arsip backup lokal | `1000M` atau lebih | Jumlah part lebih sedikit kalau tidak tergantung jaringan. |

Kalau bingung mulai dari mana, pakai **`100M`** dulu. Itu biasanya jadi titik aman paling praktis.

## #2 Proses Membagi File Menjadi Beberapa Part RAR

Setelah WinRAR terinstal, ikuti langkah berikut.

<Steps>
   <Step>
      **Siapkan file atau folder target**
      Tentukan file atau folder yang ingin dibagi menjadi beberapa part.
   </Step>

   <Step>
      **Buka menu arsip WinRAR**
      Klik kanan file/folder, lalu pilih **Add to archive...**.
   </Step>

   <Step>
      **Atur konfigurasi split volume**
      Di jendela pengaturan, fokus ke bagian berikut:
      - **Archive name:** Ubah nama file output jika diperlukan.
      - **Archive options:** Centang **Create solid archive** dan **Add recovery record** untuk menjaga integritas data.
      - **Split to volumes, size (paling penting):** Tentukan ukuran tiap part. Bisa pilih preset atau ketik manual, misalnya `100M` untuk 100 MB per part.
   </Step>

   <Step>
      **Jalankan proses split**
      Klik **OK** lalu tunggu sampai selesai. Hasilnya akan muncul dalam beberapa file seperti `.part1.rar`, `.part2.rar`, dan seterusnya.
   </Step>
</Steps>

![Jendela pengaturan WinRAR yang menampilkan konfigurasi split file menjadi beberapa volume.](/images/_posts/apps/winrar/split_rar.webp "Pengaturan split to volumes di WinRAR pada Windows.")
*Pengaturan utama ada di opsi **Split to volumes, size**.*

## #3 Cara Menggabungkan dan Mengekstrak Kembali

Supaya file bisa diekstrak sempurna di sisi penerima, ada satu aturan utama yang wajib dijelaskan.

### Semua Part Harus Ada di Satu Folder

Semua file part harus berada di folder yang sama, contohnya:
- `file.part1.rar`
- `file.part2.rar`
- `file.part3.rar`

Kalau satu part saja hilang, proses ekstraksi akan gagal.

### Langkah Ekstraksi

<Steps>
   <Step>
      **Pastikan semua part ada dalam satu folder**
      Cek ulang bahwa semua file part sudah lengkap dan berada di direktori yang sama.
   </Step>

   <Step>
      **Mulai ekstraksi dari file pertama**
      Klik kanan `part1.rar`, lalu pilih **Extract Here** atau **Extract to...**.
   </Step>

   <Step>
      **Biarkan WinRAR menggabungkan otomatis**
      WinRAR akan membaca part berikutnya secara otomatis sampai proses ekstraksi selesai.
   </Step>
</Steps>

Selesai. Simpel, praktis, dan sangat berguna untuk menangani file berukuran besar.

## Masalah yang Paling Sering Terjadi (dan Solusinya)

Kalau proses ekstraksi gagal, biasanya penyebabnya salah satu dari ini:

1. **Ada part yang belum terunduh lengkap**
   Cek ulang daftar file part, lalu unduh ulang part yang hilang.
2. **Salah satu part rusak/corrupt**
   Cukup unduh ulang file part yang rusak, tidak perlu semua part.
3. **Ekstraksi dimulai dari file yang salah**
   Selalu mulai dari `part1.rar`, bukan `part2` dan seterusnya.
4. **Nama file part berubah**
   Pastikan urutan nama file asli tidak diubah agar WinRAR bisa membaca rantainya.

## Validasi Sebelum Dibagikan (Opsional tapi Disarankan)

Supaya penerima tidak dapat error “archive corrupt”, lakukan validasi sederhana ini:

<Steps>
   <Step>
      **Tes ekstraksi di komputer sendiri**
      Coba ekstrak hasil split sekali sebelum kamu kirim ke orang lain.
   </Step>

   <Step>
      **Pisahkan folder arsip aktif dari arsip lama**
      Hindari mencampur file part baru dengan file uji coba lama yang namanya mirip.
   </Step>

   <Step>
      **Aktifkan recovery record saat membuat arsip**
      Opsi **Add recovery record** membantu toleransi jika ada kerusakan minor pada data.
   </Step>
</Steps>

Langkah kecil ini biasanya sangat mengurangi keluhan gagal ekstrak dari penerima file.
