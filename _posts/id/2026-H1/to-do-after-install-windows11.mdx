## Bagian 1: Langkah Awal yang Krusial

Tiga langkah pertama ini wajib agar sistem aman, stabil, dan tampil dengan pengaturan yang tepat.

### #1 Cek Update Windows dan Microsoft Store

Ini langkah paling penting karena akan menarik patch keamanan terbaru, perbaikan bug, dan driver esensial.

1. Buka **Settings > Windows Update** (atau tekan <kbd>Win</kbd> + <kbd>I</kbd>).
2. Klik **Check for updates** lalu instal semua update yang tersedia.
3. Buka **Microsoft Store**, masuk ke **Library**, lalu klik **Get updates** untuk memperbarui aplikasi bawaan.

![Halaman Windows Update saat memeriksa update sistem dan patch keamanan terbaru.](/images/_posts/windows11/to-do-after-install-windows11/1-windows-update.webp "Windows Update di Windows 11.")

### #2 Atur Resolusi dan Skala Layar

Setelah instalasi baru, kadang Windows belum menerapkan pengaturan tampilan paling nyaman untuk monitor kamu.

1. Klik kanan Desktop > **Display settings**.
2. Atur **Display resolution** ke nilai tertinggi yang direkomendasikan.
3. Cek **Scale**. Kalau 150% terasa terlalu besar, coba **125%** atau **100%**.

![Pengaturan tampilan Windows 11 dengan fokus pada opsi resolusi dan skala layar.](/images/_posts/windows11/to-do-after-install-windows11/2-display-optimasi.webp "Pengaturan optimasi display Windows 11.")

### #3 Aktifkan Dark Mode Otomatis

Windows 11 belum menyediakan dark mode otomatis berbasis waktu secara lengkap. Untuk kenyamanan mata, terutama malam hari, gunakan aplikasi pihak ketiga yang tepercaya.

- **Aplikasi:** Auto Dark Mode
- **Deskripsi:** Mengubah tema terang dan gelap otomatis berdasarkan waktu matahari terbit dan terbenam.
- **Baca panduan & Download:** [Guide to Setting Up Automatic Dark Mode in Windows 11](/blog/auto-dark-mode-windows-11)

## Bagian 3: Optimasi Fitur Bawaan Windows

Aktifkan fitur bawaan berikut supaya pengalaman pakai harian lebih nyaman.

- **Phone Link:** Hubungkan Android atau iPhone ke PC untuk notifikasi dan balas pesan langsung dari desktop.
- **Aktifkan Quick End Task:** Klik kanan taskbar > **Taskbar settings** > aktifkan **End task**.

![Pengaturan taskbar Windows 11 dengan opsi End task dalam keadaan aktif.](/images/_posts/windows11/to-do-after-install-windows11/end_task.webp "Aktifkan End task di taskbar Windows 11.")

- **Kelola Startup Apps:** Buka **Task Manager** (<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Esc</kbd>), masuk tab **Startup apps**, lalu nonaktifkan aplikasi startup yang tidak perlu.
- **Aktifkan OneDrive:** Pastikan kamu login OneDrive agar file penting dibackup otomatis ke cloud.
- **Gunakan Sticky Notes:** Simpel tapi sangat membantu untuk catatan cepat.

Kalau semua langkah di atas sudah dilakukan, Windows 11 kamu bukan cuma fresh, tapi juga siap dipakai produktif dengan lebih stabil dan nyaman.
