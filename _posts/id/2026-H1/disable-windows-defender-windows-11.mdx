Microsoft Defender adalah solusi antivirus bawaan yang tangguh di Windows 11. Namun, ada kalanya Anda perlu menonaktifkannya untuk sementara—misalnya saat menginstal perangkat lunak atau game yang sah namun sering keliru terdeteksi sebagai ancaman (*false positive*), atau untuk tujuan *troubleshooting*.

Panduan ini akan menunjukkan cara yang benar dan aman untuk mematikan perlindungan *real-time* di Windows 11, berlaku untuk semua versi pembaruan utama termasuk 22H2, 23H2, dan 24H2.

## Penting: Pahami Risikonya!

Menonaktifkan antivirus, bahkan untuk sementara, akan membuat PC Anda lebih rentan terhadap *malware*, virus, dan ancaman lainnya. Lakukan ini hanya jika Anda benar-benar yakin dengan file atau aplikasi yang akan Anda jalankan. Setelah selesai, segera aktifkan kembali perlindungan tersebut.

## Kapan Anda Mungkin Perlu Mematikan Defender?

Meskipun tidak disarankan untuk penggunaan sehari-hari, beberapa skenario yang wajar untuk mematikan antivirus sementara adalah:

- **Instalasi Game/Software Tertentu:** Beberapa *installer* atau file modifikasi game (untuk penggunaan yang sah tentunya) sering dimodifikasi dan dapat memicu deteksi antivirus.
- **Menggunakan Alat Sistem:** Aplikasi untuk *resetter* printer atau aktivasi sistem kadang memerlukan akses tingkat rendah yang diblokir oleh Defender.
- **Troubleshooting Konflik:** Untuk memeriksa apakah antivirus adalah penyebab suatu aplikasi berjalan lambat atau mengalami error seperti `0xc000007b`.

## Panduan Menonaktifkan Perlindungan Real-time

Perlu diingat, Windows akan secara otomatis mengaktifkan kembali perlindungan ini setelah beberapa saat atau setelah *restart* untuk menjaga keamanan sistem Anda.

### Langkah 1: Buka Keamanan Windows (Windows Security)

Cara termudah adalah melalui menu Start.

1. Klik menu **Start**.
2. Ketik "Windows Security" atau "Keamanan Windows".
3. Klik aplikasi yang muncul di hasil pencarian.

![Membuka aplikasi Windows Security dari menu Start](/images/_posts/windows11/antivirus-disable/ms-defender-0.webp)
*Membuka aplikasi Windows Security dari menu Start.*

### Langkah 2: Masuk ke Pengaturan "Virus & threat protection"

Di dalam dasbor Keamanan Windows, pilih menu utama di sisi kiri atau ikon perisai.

1. Klik **Virus & threat protection**.

![Dasbor Windows Security dengan menu Virus and threat protection disorot](/images/_posts/windows11/antivirus-disable/ms-defender-1.webp)
*Dasbor Windows Security dengan menu 'Virus & threat protection' disorot.*

### Langkah 3: Kelola Pengaturan (Manage Settings)

Di bawah judul "Virus & threat protection settings", Anda akan menemukan tautan untuk mengelola pengaturan.

1. Klik **Manage settings**.

![Tautan Manage settings di bawah bagian Virus and threat protection settings](/images/_posts/windows11/antivirus-disable/ms-defender-2.webp)
*Tautan 'Manage settings' di bawah bagian 'Virus & threat protection settings'.*

### Langkah 4: Matikan "Real-time protection"

Ini adalah saklar utama untuk perlindungan aktif.

1. Temukan saklar di bawah **Real-time protection**.
2. Klik saklar tersebut untuk mengubahnya dari **On** menjadi **Off**.
3. Sebuah jendela User Account Control (UAC) akan muncul untuk meminta konfirmasi. Klik **Yes**.

![Saklar Real-time protection dalam posisi Off di Windows 11](/images/_posts/windows11/antivirus-disable/ms-defender-4.webp)
*Saklar 'Real-time protection' dalam posisi Off.*

Pada titik ini, perlindungan antivirus *real-time* Anda telah dinonaktifkan sementara. Ikon Keamanan Windows di taskbar mungkin akan menampilkan tanda seru kuning sebagai pengingat.

## Jangan Lupa Mengaktifkannya Kembali!

Setelah Anda selesai dengan tugas yang memerlukan antivirus nonaktif, sangat penting untuk mengaktifkannya kembali. Cukup ulangi langkah-langkah di atas dan klik kembali saklar **Real-time protection** ke posisi **On**.

Dengan mengikuti panduan ini, Anda bisa mengelola keamanan sistem Anda dengan lebih fleksibel tanpa mengorbankan perlindungan dalam jangka panjang.
