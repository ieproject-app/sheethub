Performing a clean install of Windows 11 is the ideal way to get a stable, lightweight system free from unnecessary bloatware. The most effective method for this is using a bootable USB installation media.

![A USB flash drive ready to be used.](/images/_posts/rufus/rufus-banner.webp)

One of the most reliable and popular utilities for this task is **Rufus**—a lightweight, fast, and highly flexible application, including the ability to customize installation requirements like TPM 2.0 and Secure Boot.

This guide will explain the entire process systematically, from the preparation stage until the bootable USB is ready for installation.

## 2. Creating the Bootable Windows 11 Drive

Once all preparations are complete, follow these steps.

### Step 1 — Run Rufus and Select the Device
Open the Rufus application. Connect the flash drive to your computer, and the device will be automatically detected under the **Device** section.

![Initial Rufus screen with the flash drive detected](/images/_posts/rufus/rufus-4.9_1.png)

Ensure the correct device is selected before continuing.

### Step 2 — Select the Windows 11 ISO File
In the **Boot selection** section, click **SELECT**, then navigate to and choose the Windows 11 ISO file you downloaded earlier.

![Selecting the Windows 11 ISO file in Rufus](/images/_posts/rufus/rufus-4.9_2.png)

After selecting the file, Rufus will automatically adjust some other parameters.

### Step 3 — Verify the Partition Scheme
For installation on modern systems, ensure the following configuration is used:
- **Partition scheme:** GPT
- **Target system:** UEFI (non-CSM)

### Step 4 — Start the Process
Click the **START** button to begin creating the bootable media.

### Step 5 — Configure Windows User Experience
After pressing START, an additional configuration window will appear. Here you can bypass some of the Windows 11 installation requirements.

Check the **Remove requirement for 4GB+ RAM, Secure Boot and TPM 2.0** option if your computer does not meet these specifications.

![Windows customization window in Rufus](/images/_posts/rufus/rufus-4.9_3.png)

Click **OK** to proceed. Rufus will provide a final warning that all data on the flash drive will be erased. Click OK again to confirm.

### Step 6 — Wait for Completion
Rufus will now start copying the installation files to the flash drive. This process usually takes a few minutes.

![The process of creating the bootable USB is complete with a READY status](/images/_posts/rufus/rufus-4.9_4.png)

When the status bar is full and displays **READY**, your bootable flash drive is ready to use.
