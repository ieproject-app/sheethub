Performing a clean installation of Windows 11 is the right step to get a stable, lightweight system free from unnecessary built-in applications. The most effective method to do this is by using a USB-based installation media.

One of the most reliable utilities for this task is **Rufus** — a lightweight, fast, and flexible application, including the ability to customize installation requirements like TPM 2.0 and Secure Boot.

This guide will systematically explain the entire process, from the preparation stage until the bootable USB is ready to use.

---

## 1. Preparation Stage (Required)

Before starting the process of creating the installation media, make sure the following components are available:

### 1.1. USB Flash Drive

Use a flash drive with a minimum capacity of **8 GB**. Ensure the device is in good condition.

### 1.2. Windows 11 ISO File

Prepare the Windows 11 installer file in `.iso` format.

<DownloadButton id="windows-11-iso-24h2" />

### 1.3. Rufus Application

Use the latest version of Rufus.

<DownloadButton id="rufus-installer" />

### 1.4. Back Up Important Data

> ⚠️ **Warning**
> This process will completely format the flash drive. All data on it will be permanently deleted.

---

## 2. Creating the Bootable Windows 11 Media

![USB Flash Drive plugged into a laptop](/images/_posts/rufus/rufus-banner.webp)

Once all preparations are complete, proceed with the following steps.

---

### Step 1 — Run Rufus and Select the Device

Open the Rufus application and connect the flash drive to the computer. The device will be automatically detected under the **Device** section.

![Initial Rufus screen with flash drive detected](/images/_posts/rufus/rufus-4.9_1.png)

Make sure the correct device is selected before proceeding.

---

### Step 2 — Select the Windows 11 ISO File

In the **Boot selection** section, click **SELECT**, then choose the downloaded Windows 11 ISO file.

![Selecting the Windows 11 ISO file in Rufus](/images/_posts/rufus/rufus-4.9_2.png)

Once the file is selected, Rufus will automatically configure several parameters.

---

### Step 3 — Verify Partition Scheme

Ensure the following configuration is used:

-   **Partition scheme:** GPT
-   **Target system:** UEFI (non-CSM)

This configuration is standard for installing Windows 11 on modern systems.

---

### Step 4 — Start the Process

Click the **START** button to begin creating the bootable media.

---

### Step 5 — Configure Windows User Experience

After pressing START, an additional configuration window will appear.

Check the following option if needed:

-   **Remove requirement for 4GB+ RAM, Secure Boot and TPM 2.0**

![Windows customization window in Rufus](/images/_posts/rufus/rufus-4.9_3.png)

Click **OK** to continue the process.

---

### Step 6 — Wait for Completion

Rufus will start copying the installation files to the flash drive. This process takes a few minutes.

![Process completed with READY status](/images/_posts/rufus/rufus-4.9_4.png)

When the status changes to **READY**, your flash drive is ready for use.

---

## Conclusion

The Windows 11 installation media is now ready to be used for a clean installation on UEFI-based devices. With the right configuration, the installation process will be more efficient and encounter fewer issues.
