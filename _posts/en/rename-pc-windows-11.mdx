If your fresh Windows 11 install shows a random name like `DESKTOP-CFL3R1S`, you can rename it in less than 2 minutes: open **Settings > System > About**, click **Rename this PC**, enter your preferred name, then restart. After reboot, the new name appears in your network list, Microsoft account devices page, and local system info.

This is one of the first tweaks worth doing after setup because a clear device name reduces confusion when you manage multiple PCs.

## Why Renaming Your PC Matters

A good PC name helps in daily admin tasks, especially when you:

- identify devices in **home/office Wi-Fi**,
- map shared folders on LAN,
- review devices in your Microsoft account,
- troubleshoot remote connections.

If you also plan to share files on local network, continue with this guide after renaming: [How to Share a Folder in Windows 11 via LAN and WiFi](/blog/share-folder-windows-11-lan-wifi).

## Prerequisites Before You Rename

- You are signed in with an account that has **administrator** permission.
- Save your active work, because a **restart is required** to fully apply the new name.
- Use a short, clear naming format (example: `OFFICE-EDIT-01`, `LAPTOP-RINA`, `DESKTOP-STUDIO`).

<Callout variant="info" title="Freshness Note">
  Verified for Windows 11 23H2 and 24H2 as of March 2026. Microsoft may adjust UI labels in future builds, but the path remains under System settings.
</Callout>

## Step-by-Step: Rename PC in Windows 11

<Steps>
  <Step>
    **Open System settings quickly**

    Right-click **Start**, then click **System**.

    ![Right-click Start menu in Windows 11 and choose System](/images/_posts/windows11/rename/rename-pc1.webp "Open System page from Start menu")
    *Fastest path: Start right-click -> System.*
  </Step>

  <Step>
    **Click "Rename this PC"**

    On the **System > About** page, find the **Rename this PC** button near your current device name and click it.

    ![Windows 11 System About page with Rename this PC button highlighted](/images/_posts/windows11/rename/rename-pc2.webp "Locate Rename this PC on System About")
    *You will see the current name at the top of this page.*
  </Step>

  <Step>
    **Enter your new name**

    Type the new computer name, then click **Next**.

    ![Rename your PC popup in Windows 11 with new device name field](/images/_posts/windows11/rename/rename-pc3.webp "Enter the new PC name")
    *Use a consistent pattern if you manage more than one machine.*
  </Step>

  <Step>
    **Restart to apply**

    Choose **Restart now** (or **Restart later** if you need to save work first).

    ![Windows 11 restart prompt after renaming PC](/images/_posts/windows11/rename/rename-pc4.webp "Restart required to apply new PC name")
    *The rename process is complete only after reboot.*
  </Step>
</Steps>

## Verify the New Name (Quick Checks)

After restart, run these checks so you know the rename actually applied everywhere.

### Check 1: Settings page

Go to **Settings > System > About** and confirm the displayed device name.

### Check 2: Command output

Open Command Prompt and run:

```powershell
hostname
```

Expected result:

```text
OFFICE-EDIT-01
```

(Your output should match your chosen name.)

### Check 3: Network visibility

From another PC in the same network, verify the new name appears when browsing available devices.

## Common Rename Issues and Fast Fixes

### 1) Name changed in Settings, but network still shows old name

- Restart one more time.
- Power-cycle the router or reconnect to Wi-Fi.
- Flush cache on the client PC by reopening File Explorer network view.

### 2) Rename button is blocked or unavailable

- Sign in with an administrator account.
- If this is a company-managed device, check policy restrictions with your IT admin.

### 3) New name rejected by Windows

- Remove unusual symbols.
- Keep the name short and practical.
- Use letters, numbers, and hyphens for best compatibility.
