I've been using SketchUp for home renovation planning for a while now — from estimating material needs to visualizing room layouts. It's been genuinely helpful. But as I'm not a professional designer, paying for a Pro subscription always felt like a bit of an overkill.

After doing some research, I found that **SketchUp 2017 Make** is the last officially released version that is both free and works fully offline. This is exactly why it's still so highly sought after today — especially for personal or non-commercial projects.

<Callout variant="warning" title="Important Note on Licensing">
  The **Make** version is for **non-commercial** use only — such as hobbies, personal projects, or education. If you are using SketchUp for professional work that generates income, you are required to use the **Pro** license. SketchUp 2017 Make was the last free version before Trimble shifted to web-based and subscription models.

## First Impressions

When I first installed it, I was greeted by a 30-day trial notification — which made me a bit hesitant at first. Was it truly free, or just a trial period before I had to reach for my wallet?

*The SketchUp Make 2017 trial screen — it looks just like the paid version, but there is a nice surprise after the 30 days are up.*

> **Update after 30 days:** Once the trial period ends, the application actually continues to work normally under the **SketchUp Make 2017 User** license. The functionality remains intact for non-commercial needs, so there is no need to worry.
